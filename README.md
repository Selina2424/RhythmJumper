# RhythmJumper
A Unity rhythm platformer built in C# featuring adaptive difficulty, beat synchronised gameplay and reaction time analysis.

## Features

- Beat synchronised gameplay
- Adaptive difficulty
- Reaction time testing
- Performance tracking
- Rhythm accuracy analysis
- Modular game architecture

## Technologies

- Unity
- C#
- Visual Studio

## Controls

- A / D - Move
- Space - Jump

## Future Improvements

- Additional levels
- More music tracks
- Leaderboard
- Improved visual effects
- Save system
